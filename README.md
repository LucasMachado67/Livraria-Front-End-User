
# Livraria E-commerce Front-End (user)

Neste projeto, está a parte onde o usuário poderá acessar, consultar os livros disponíveis, criar uma conta e efetuar o login.

![Angular](https://img.shields.io/badge/Angular-17.3.0-red)

![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3.3-blue)



## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/usuario/nome-do-repositorio.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd nome-do-repositorio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Para rodar a aplicação:
   ```bash
   ng serve
   ```

5. Acesse a aplicação no navegador em:
   [http://localhost:4200](http://localhost:4200)
